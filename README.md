# 🤖 Smart Query AI-Based Chatbot

An intelligent chatbot powered by **OpenAI GPT-4o-mini**, designed to deliver human-like responses, perform live web searches, and provide accurate solutions to user queries. This project demonstrates seamless integration of OpenAI’s API with a custom frontend/backend interface for a rich conversational experience.

---

## 🚀 Features
- 💬 **Natural Conversations** – Chat interface for smooth human-AI interaction  
- 🔍 **Web Search Integration** – Fetches live information in real time  
- 📚 **Contextual Responses** – Provides helpful and accurate answers  
- ⚙️ **Lightweight & Fast** – Uses GPT-4o-mini for efficient inference  
- 🔐 **Secure API Access** – Environment variable support for safety  
- 📱 **Responsive UI** – Works across devices (frontend included)  

---

## 🛠 Tech Stack
| Layer       | Technology |

## ⚡ Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/Devashish043/smart-query-AI_based_chatbot.git
cd smart-query-AI_based_chatbot
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Set Up Environment Variables
Create a `.env` file and add your OpenAI API key:
```bash
OPENAI_API_KEY=your_api_key_here
```

### 4. Run the Project
```bash
npm run dev
```
